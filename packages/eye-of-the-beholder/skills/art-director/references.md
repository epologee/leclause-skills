# References

Sources for the three modules in SKILL.md. Each work summarized in one sentence, with ISBN where applicable and URL where freely available. Grounded in the brand-identity, typography, and design-system canon.

## Module 1: Brand identity discovery

### Designing Brand Identity

Alina Wheeler, *Designing Brand Identity: An Essential Guide for the Whole Branding Team* (Wiley, 6th edition, 2017). ISBN 978-1-119-35386-7. The canonical workbook for brand identity development, structured in a five-phase model (research, strategy, design, touchpoints, management) that art-director's Module 1 follows.

### The Brand Gap

Marty Neumeier, *The Brand Gap: How to Bridge the Distance Between Business Strategy and Design* (New Riders, revised second edition, 2005). ISBN 978-0-321-34810-9. Compact essay-book on why brand is more than logo, with the "brand as onion" model (functional, emotional, self-expressive) used in 1.2 for the promise definition.

### Logo Design Love

David Airey, *Logo Design Love: A Guide to Creating Iconic Brand Identities* (New Riders, second edition, 2014). ISBN 978-0-321-98520-6. Practical decision framework for logo work (favicon test, monochrome test, t-shirt test) + extensive case studies of well-known identities.

## Module 2: Visual language translation

### Type Matters!

Jim Williams, *Type Matters!* (Merrell, 2012). ISBN 978-1-85894-567-6. Pragmatic micro-typography workbook: how to make a text block readable, when to space, what eyes look for in a line. No theory, only working rules.

### Thinking with Type

Ellen Lupton, *Thinking with Type: A Critical Guide for Designers, Writers, Editors, and Students* (Princeton Architectural Press, second edition, 2010). ISBN 978-1-56898-969-3. The canonical introduction to typographic theory and history, with workbook sections. Standard course reading at most design schools.

### Interaction of Color

Josef Albers, *Interaction of Color* (Yale University Press, 50th anniversary edition, 2013). ISBN 978-0-300-17935-8. The classic work on how color changes in context, with exercises in which the same color next to different neighbors appears to be a different color. Foundation for how Module 2 thinks about color.

### The Designer's Dictionary of Color

Sean Adams, *The Designer's Dictionary of Color* (Abrams, 2017). ISBN 978-1-4197-2207-4. Per-color essay + cultural context + usage by iconic brands. Useful reference when documenting a brand hue and looking for a physical reference.

### Designing for Emotion

Aarron Walter, *Designing for Emotion* (A Book Apart, 2011). ISBN 978-1-937557-00-3. Short but influential essay on how visual design evokes emotion, structured around Maslow's pyramid. Bridge between brand personality and concrete visual choices.

### Practical Typography

Matthew Butterick, *Butterick's Practical Typography* (online, free). URL: https://practicaltypography.com. Opinion-driven web-only book with strict rules for body type (point size, line-height, measure). Especially strong for typographic micro-decisions in UI and document work.

### Klim Type Foundry

Klim Type Foundry, case studies and type specimens. URL: https://klim.co.nz. Shows per type family how a type designer works for specific brands, including the decision logic behind a release. Useful as a portfolio of "what does brand-ready type look like".

### Fontsmith / Monotype Studio

Fontsmith (now part of Monotype Studio), case studies and brand type work. URL: https://www.monotype.com/studio/fontsmith. Long-established brand type foundry that publicly documents how they make type for brands. Useful for the "when custom, when off-the-shelf" question in 2.1.

## Module 3: Design system architecture

### Atomic Design

Brad Frost, *Atomic Design* (self-published, 2016). ISBN 978-0-9983379-0-1. Also free online: https://atomicdesign.bradfrost.com. Presents the five-layer taxonomy (atoms, molecules, organisms, templates, pages) used in 3.2 for component organization.

### Design Systems

Alla Kholmatova, *Design Systems: A Practical Guide to Creating Design Languages for Digital Products* (Smashing Magazine, 2017). ISBN 978-3-945749-58-4. Focuses on how to design a design system as a language, with component catalog + pattern library + governance. Source for the semver-for-design-tokens approach in 3.4.

### Design Systems Handbook

Marco Suarez, Jina Anne, Katie Sylor-Miller, Diana Mounter, and Roy Stanfield, *Design Systems Handbook* (DesignBetter / InVision, 2017, free). URL: https://www.designbetter.co/design-systems-handbook. Free online book that describes design-system teams as a product: roadmaps, governance, communication, rollout. Complement to Kholmatova for team organization.

### Expressive Design Systems

Yesenia Perez-Cruz, *Expressive Design Systems* (A Book Apart, 2019). ISBN 978-1-952616-09-0. Pushback against the sterile component-library view. How to keep expressiveness and brand personality alive within the systematics of a design system. Source for 3.5 living docs.

### Refactoring UI

Adam Wathan and Steve Schoger, *Refactoring UI* (self-published, 2018). URL: https://www.refactoringui.com. Tactical rules for specific UI problems: visual hierarchy, color choice, spacing, text styling. Complement to Atomic Design: Frost gives the structure, Wathan/Schoger give the moves within that structure.

### Eightshapes (Nathan Curtis)

Nathan Curtis, essays on Eightshapes Medium publication. URL: https://medium.com/eightshapes-llc. Deep technical essays on token taxonomy (the 3-layer primitive / semantic / component model from 3.1), contribution models, DS team structures, and the role of the DS lead. Primary source for Module 3's architecture choices.

## Public design systems as reference

Not to copy, but to study patterns. Five large public systems with excellent documentation:

- **Polaris** (Shopify): https://polaris.shopify.com. Strong in voice and content tone.
- **Carbon** (IBM): https://carbondesignsystem.com. Strong in token taxonomy and a11y.
- **Primer** (GitHub): https://primer.style. Strong in developer ergonomics and contribution model.
- **Material 3** (Google): https://m3.material.io. Strong in theming and dynamic color.
- **Atlassian Design System**: https://atlassian.design. Strong in governance and multi-product rollout.

## Canon (general)

### The Vignelli Canon

Massimo Vignelli, *The Vignelli Canon* (Lars Muller, 2010). Vignelli himself once released a free PDF; the official hosting source is no longer stable, search via the publisher or Internet Archive. Strict manifesto on discipline and deliberate restriction of choices in visual design. Aligns with art-director's stance that recording choices upfront is better than re-choosing per feature.
